Windows命令提示符 CMD 常用操作

YYT6801 发布于 2025-09-30 收录于 编程开发

常用命令

  • 切换盘符 cd 盘符: 也可直接输入盘符切换盘符，如：E:
  • 切换到当前路径的上一级目录
    cd ..
  • 切换到当前路径的下一级目录
    cd 目录名
  • 切换到当前路径的上一级目录的上一级目录
    cd ../..
  • 切换到当前路径
    cd /d %~dp0
  • 清空剪切板
    cmd /c "echo off | clip
    或
    dir|clip
