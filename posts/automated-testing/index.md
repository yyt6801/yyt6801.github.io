# 自动化测试入门


## 测试金字塔
- 单元测试 → 集成测试 → E2E测试
- 各层级比例与关注点

## 单元测试
### Python示例（pytest）
- 断言写法
- Fixture与参数化
- Mock与打桩

### 测试覆盖率
- 覆盖率指标解读
- 追求合理的覆盖率而非100%

## 集成测试
- 数据库测试（内存数据库 vs 测试容器）
- API接口测试

## CI中集成测试
- GitHub Actions中运行测试
- 测试报告与可视化
- 门禁机制（测试不通过禁止合并）

## 工业软件测试的特殊性
- 硬件在环测试（HIL）
- 仿真环境测试
- 协议一致性测试
